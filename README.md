# TaxiGo 🚖

**TaxiGo** is a modern full-stack taxi booking application for Morocco. Users can book taxis quickly and securely, with verified drivers, transparent pricing, and free cancellations.  

The project uses **Laravel** for the backend (API & database management) and **React + Tailwind CSS** for the frontend (responsive, modern UI).

## Features

- Real-time taxi booking across Morocco.
- Verified drivers for secure rides.
- Transparent pricing, no hidden fees.
- Free ride cancellation.
- Responsive design, mobile-friendly.
- User authentication and profile management.
- Modern animations and interactive UI.

---

## Tech Stack

**Frontend:**
- React
- Tailwind CSS
- Vite
- React Router

**Backend:**
- Laravel 10
- PHP 8+
- MySQL / MariaDB
- Sanctum (API authentication)

**Deployment:**
- Frontend: Vercel, Netlify, or any static hosting
- Backend: Laravel server (e.g., DigitalOcean, Render)
